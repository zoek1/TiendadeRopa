Proyecto de Programación

Este proyecto pretende implementar cada uno de los elementos aprendidos durante el presente de Programación orientada a objetos como son:

  • Herencia
  • Polimorfismo
  • Clases abstractas
  • Interfaces
  • Interfaces gráficas

Haciendo uso del lenguaje de programación Java.

Vulnerabilidades

  • La base de datos esta en texto plano.
  • La suma de verificación md5 puede ser agregada fácilmente en el archivo de contraseñas.
  • Cuando se agrega un producto, una " , " distorsionaría la información.
  • En el campo Precio y Descuento de la clase ProductoGui, si se ingresan caracteres se aplica un cast y la suma no sera correcta.

La documentación de clases podrá ser encontrada en la wiki de Tienda de ropa. En esta se ha trabajado para que cada constructor, método y constantes codificado sea descrita con un estilo parecido a la documentación de Oracle.

El Proyecto puede ser compilado de dos maneras:

Compilación 1:

Compilando con una llamada hacia make desde la consola, cabe destacar que se necesita tener tener instalado make.

Tiendaderopa@axelfrog $ make

Para ejecutar el programa se hace igual de dos maneras escritas desde la consola con:

Ejecutar 1:

Tiendaderopa@axelfrog $ java -jar TiendadeRopa.jar

Ejecutar 2:

Tiendaderopa@axelfrog $ java Main

Compilación 2:

En esta opción solo se necesita compilar el archivo principal:

Tiendaderopa@axelfrog $ javac Main.java

Para ejecutar solo se necesita ejecutar el archivo principal:

Tiendaderopa@axelfrog $ java Main

El proyecto puede ser descargado desde las dos etiquetas en la parte superior de esta pagina o la versión 1.0 desde este link Tienda de Ropa 1.0

Al dar click a la tercera etiqueta en la parte superior les mostrara todo la actividad del proyecto.
